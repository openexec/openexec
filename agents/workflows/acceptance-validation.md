<instructions>
Check implementation against acceptance criteria.
</instructions>

<process>
1. Read acceptance criteria from story file
2. For each criterion:
   - Identify which tests verify this criterion
   - Confirm tests are passing
   - Manually verify behavior if needed
3. Document validation results
4. If any criteria not met:
   - Identify gap
   - Plan additional test/implementation
   - Execute red-green cycle
5. Confirm all acceptance criteria satisfied
</process>
