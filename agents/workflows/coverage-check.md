<instructions>
Verify test coverage meets project standards.
</instructions>

<process>
1. Run test coverage tool
2. Review coverage report
3. Identify uncovered code paths
4. Assess whether uncovered code is:
   - Dead code (remove it)
   - Edge cases needing tests (add them)
   - Defensive code (acceptable gaps)
5. Add tests for meaningful gaps
6. Confirm coverage meets project thresholds
</process>
